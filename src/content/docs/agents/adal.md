## What It Does

AdaL CLI is a self-evolving coding agent that runs in both terminal and web interfaces. Built on the [AdalFlow](https://github.com/SylphAI-Inc/AdalFlow) library, it features multi-model support, MCP tool integration, skills system, and the ability to improve its own capabilities over time.

## Key Strengths

- **Self-evolving**: Learns from interactions and improves its own tooling
- **Dual interface**: Terminal CLI and browser-based web UI
- **MCP-native**: Full Model Context Protocol support
- **Skills system**: Extensible capabilities via SKILL.md files
- **Multi-model**: Supports Claude, GPT, Gemini, and local models
- **AdalFlow library**: Open-source agentic AI framework
- **Research-backed**: Incorporates state-of-the-art AI agent research

## Built on AdalFlow

AdalFlow is the open-source library powering AdaL CLI:
- Modular agent architecture
- Prompt optimization and caching
- Tool orchestration
- Memory management
- Multi-provider model support

## Best For

- Developers who want a customizable, extensible agent
- Researchers exploring agentic AI
- Projects that need both terminal and web access
