## What It Does

GitHub Copilot is the most widely adopted AI coding assistant. It provides inline completions, chat, and an agent mode that can autonomously work on coding tasks within VS Code and other IDEs.

## Key Strengths

- **Ubiquity**: Available in VS Code, JetBrains, Neovim, and more
- **GitHub integration**: Understands issues, PRs, and your repo structure
- **Agent mode**: Autonomous coding within VS Code
- **Code review**: AI-powered PR reviews
- **Extensions**: Growing ecosystem of Copilot extensions
