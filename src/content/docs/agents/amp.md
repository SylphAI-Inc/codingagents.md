## What It Does

Amp is Sourcegraph's AI coding agent that combines deep codebase search with agentic coding capabilities. It's one of the first agents to natively support the ACP protocol.

## Key Strengths

- **Codebase search**: Powered by Sourcegraph's code intelligence
- **ACP-native**: Works in any ACP-compatible editor
- **Multi-repo**: Can search and work across repositories

## Best For

- Large codebases with complex dependencies
- Teams using Sourcegraph for code search
- ACP-compatible editor users (Zed, JetBrains)
