## Overview

Microsoft Agent Framework unifies AutoGen (multi-agent conversations) and Semantic Kernel (AI orchestration) into a single SDK. It supports building multi-agent workflows with MCP integration, targeting enterprise developers on both Python and .NET.

## Key Features

- **Multi-agent conversations** — Agents communicate via messages
- **MCP support** — Native Model Context Protocol integration
- **Dual runtime** — Python and .NET support
- **Azure integration** — Deep Azure AI and Microsoft 365 integration
- **Semantic Kernel** — AI orchestration and planning

## When to Use

✅ Enterprise .NET stack
✅ Azure ecosystem
✅ Need multi-agent conversation patterns
✅ Microsoft 365 integration
