## Overview

Pydantic AI brings type safety to AI agents. Built by the Pydantic team, it provides a clean, model-agnostic API for building agents with structured outputs, dependency injection, and streaming — with full type checking.

## Key Features

- **Type-safe** — Structured outputs validated by Pydantic
- **Model-agnostic** — OpenAI, Anthropic, Gemini, Groq, Mistral
- **Dependency injection** — Clean testing and modularity
- **Streaming** — Structured streaming responses
- **Logfire integration** — Built-in observability

## When to Use

✅ Production APIs needing type safety
✅ Want model-agnostic design
✅ Value clean, minimal APIs
✅ Need structured LLM outputs
